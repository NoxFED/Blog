## 固定高度宽度
``` html
<div class="wrapper">
  <div class="center">
    center
  </div>ｄｄｆ
</div>

```
``` css
.wrapper {
  width: 300px;
  height: 150px;
  position: relative;
}

.center {
  width: 100px;
  height: 50px;
  top: 50%;
  left: 50%;
  margin-left: -50px;
  margin-top: -25px;
}

```
